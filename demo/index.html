Paliari Vue

A simple lib utilities for Vue.js

Get Started

Install

$ npm install paliari-vue
or 
$ yarn add paliari-vue

Initialize

import Vue from 'vue'
import PaliariVue from 'paliari-vue'
import 'paliari-vue/dist/paliari-vue.css'
Vue.use(PaliariVue)
